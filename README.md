# auto_resize_pic
一个能对页面上的图片进行自动缩放的JS插件，避免变形。
# API

   set img class=autoResize 如下：
   <img src="1458527213330.jpg" width="100%" class="autoResize">

在页面样式中加上

  .picbox{
            width:300px;
            height:100px;
            display:block;
            overflow:hidden;
            border:1px solid red;
            position:relative;
            left:0px;
            top:0px;
        }
